# ChuckDeviceConfigurator  

## Description  
Controls devices that request jobs as well as includes a dashboard interface to configure job controllers and other required entity types.  

- Dashboard management UI  
- Device controller  
- Plugin system

## Hosted Services  

### AccountStatusHostedService  
Checks all accounts with any of the following status and marks the interested columns as `NULL` in the database if the account punishment time has lapsed.  

- `Warning`  
- `Suspended`  
- `Cooldown`
